# NHL Team Stats Web Scraping and Analysis

## Purpose
The purpose of this project is to scrape NHL team statistics from multiple pages and analyzes trends in wins, goals, and team performance across six seasons (1990–1995).

## Data Source:
- **Website:** https://www.scrapethissite.com/pages/forms/
- **Pages Scraped:** 5 pages of NHL team season stats

## Questions Answered
1. What was the average win percentage across all teams each year?
2. Which year had the highest average goals scored per team?
3. How many teams had a winning record (Win Pct > 0.5) each year?
4. What are the top 5 teams with the most wins in a single season?

## Files
- nhlscraping.ipynb: A Jupyter Notebook containing web scraping code, data cleaning and analysis, and data visualizations.
- README.md: Readme file serving as a project description.
